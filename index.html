Getting Started

There are some ways to get started. First, see all download options available below, then choose the most suitable option for your need. Incase you are looking to quickly add essentials.css to your project use CDN option (Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.).

INSTALL WITH NPM:
npm install essentials.css
USE CDN:
https://cdn.rawgit.com/svichas/essentials.css/27fd41c5/dist/essentials.min.css
